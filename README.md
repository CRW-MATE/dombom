# web game/ engine 
\__<(.<->.)>__/
this project is my attempt in making a game engine from scratch and using said engine to create a game ,all the assets and code in this project has been made by me, my thought process when doing the
techincal parts of the engine like how does the saving or displaying happen follows 3 certain guidlines, first making it run anywhere just like any other website,second avoid any third party modules,packages,databases to insure it runs everywhere,third having fun and being laxed but not at the expence of the user experience.
# index.html 
it's the webpage that displays the game ,its in querks mode ,I tried noemal mode but doesnt work.
# jystick.js
handles layout and appearance for the html that means the canvas, the joystic, the buttons...etc
also handles the joystick input events and give an appropriate outputs it as the dire variable for the game to use
# coll.js 
every thing visual inside the canvas and collision is in there and the basis for the *engine* there are classes and functions that are made to be bulding blocks for making games in the future, it covers most basic features that are found in most simple 2d games.

# localStorage 
there are 2 local storage variables,localStorage.world is the whole world rooms saved in internal storage and second is PpN that is the save you get
